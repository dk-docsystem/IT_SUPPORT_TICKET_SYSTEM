VPN-klient fejler ved opstart – kan ikke tilgå interne systemer

Beskrivelse

VPN-klienten (GlobalProtect v6.1) starter ikke efter seneste Windows-opdatering. Fejlkode 0xC000021A vises ved opstart. Brugeren kan ikke tilgå interne netværksressourcer, herunder filservere og intranet.
